========================================
01. はじめに
========================================

このMODは、World of Warships の艦長ボイスを、にじさんじEN所属のバーチャルライバー「レン・ゾット」に変更するものです。
非公式のファンメイドであり、にじさんじ及びレンさん本人には何の関係もないことをご承知おきください。


This mod changes the World of Warships captain's voice to Ren Zotto, Nijisanji EN streamer.
Please note that this is an unofficial fan-made mod and has no relation to Nijisanji or Ren himself.



========================================
02. 導入方法
========================================

この Read Me と同じフォルダにある banks を、次のフォルダ内に貼り付けてください。

	World_of_Warships > bin > xxxxxxx > res_mods

なお、xxxxxxx はゲーム本体のバージョンによって違う数字となっています。
通常、最も大きな数字が現在のバージョンに対応したフォルダなので、それを選択するようにお願いします。
貼り付けが完了したら、ゲームを起動し、音声設定の中から「音声変更」という欄を探してください。
その中に「RenZotto」という名前が存在すれば成功です。


Paste "banks" folder into the following place:

	World_of_Warships > bin > xxxxxxx > res_mods

"xxxxxxx" is a different number depending on the version of the game.
Usually, the largest number is the folder corresponding to the current version, so please select it.
After pasting is complete, start the game and look for the "Voiceover modification" field in the audio settings.
If the name "RenZotto" exists in it, you have succeeded.
